# Skill Catalog

This catalog documents reusable skill ideas and public examples. It is not an
inventory of any private machine.

| File | Purpose |
|---|---|
| [`skills.md`](skills.md) | Complete public inventory of currently known Codex, Agents, and Superpowers skills. |
| [`caveman.md`](caveman.md) | Public notes for the `caveman` example skill. |
| [`coverage-check.md`](coverage-check.md) | Latest coverage check against the local environment. |

## Catalog Rules

- Keep entries public-safe.
- Describe what a skill is for and when not to use it.
- Link to examples or templates instead of copying private runtime folders.
- Do not include secrets, private paths, memory excerpts, or live runtime state.
