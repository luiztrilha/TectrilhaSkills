# caveman

`caveman` is a compact communication skill. It reduces filler and keeps
technical content intact.

## Use When

- The user asks for shorter responses.
- Token efficiency matters.
- The work benefits from terse operational updates.

## Do Not Use When

- A destructive action needs a clear confirmation.
- Legal, security, medical, or financial warnings need full wording.
- Compression would make step order ambiguous.
- The user asks for normal prose.
